Solve a problem

This examples shows how to construct a synthetic instance of L0-regularized problem and to solve it using either the DirectSolver or the BnbSolver.

First, import all the necessary modules. The SCIP optimizer will be used in the DirectSolver.

using El0ps
using SCIP

Then, define some parameters and generate a synthetic instance.

k, m, n, ρ, s = 5, 10, 30, 0.1, 10.
x, A, y = synthetic_data_regression(k, m, n, ρ, s)

Instantiate a Least-Squares datafit and the Big-M constraint with $M=1$ as follows.

F = LeastSquares()
G = Bigm(1.)

Finally, compute a value λmax such that the all-zero vector is a solution of the problem and set the L0-regularization strength to one tenth of this value.

λmax = compute_λmax(F, G, A, y)
λ = 0.1 * λmax

Now, we are ready to construct an instance of the problem

problem = Problem(F, G, A, y, λ)

First, we solve the problem using the Branch-and-Bound solver and we enable some acceleration strategies.

solver = BnbSolver(
    dualpruning = true, 
    l0screening = true, 
    l1screening = true,
)
result = optimize(solver, problem)
println(result)

Second, we use the Direct solver to formulate the problem as a MIP. The latter is then given to the SCIP optimizer that is called with some options.

solver = DirectSolver(
    SCIP.Optimizer, 
    options = Dict(
        "display/verblevel" => 0, 
        "limits/gap"        => 1e-4,
    )
)
result = optimize(solver, problem)
println(result)
