El0ps.LeastSquares — Type
LeastSquares <: AbstractDatafit

Least-squares function

$F(\mathbf{y},\mathbf{w}) = \tfrac{1}{m} \|\mathbf{y}-\mathbf{w}\|_2^2$

where m = length(y).

source
El0ps.Logistic — Type
Logistic <: AbstractDatafit

Logistic function

$F(\mathbf{y},\mathbf{w}) = \tfrac{1}{m} \sum(\log(\mathbf{1} + \exp(- \mathbf{y} \odot \mathbf{w})))$

where m = length(y), where $\odot$ denotes the Hadamard product and where the log and the exp functions are taken component-wisely.
