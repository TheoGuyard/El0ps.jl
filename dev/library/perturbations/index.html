El0ps.Bigm — Type
Bigm <: AbstractPerturbation

Convex indicator of a Big-M constraint abs.(x) .<= h.M, where h.M > 0.

Arguments

  • M::Float64 : Big-M value.

El0ps.BigmL1norm — Type
BigmL1norm <: AbstractPerturbation

Big-M constraint plus L1-norm function h(x) = h.α * norm(x, 1) + (norm(x, Inf) < h.M ? 0. : Inf), where h.α > 0 and h.M > 0.

Arguments

  • M::Float64 : Big-M value.
  • α::Float64 : L1 regularization strength.
source

BigmL2norm <: AbstractPerturbation

Big-M constraint plus L2-norm function h(x) = h.α * norm(x, 2)^2 + (norm(x, Inf) < h.M ? 0. : Inf), where h.α > 0 and h.M > 0.

Arguments
