Problem

El0ps.Problem — Type
Problem

Problem of the form

$\min F(\mathbf{y},\mathbf{Ax}) + \lambda (\|x\|_0 + G(\mathbf{x}))$

where $\mathbf{A} \in \mathbf{R}^{m \times n}$, $\mathbf{y} \in \mathbf{R}^{m}$ and $\lambda > 0$.
